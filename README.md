# Weekly

**Weekly** is a minimalist weekly planner that helps you stay focused on what matters most — one week at a time.

## The Idea

Most task managers overwhelm you with endless backlogs, projects, and priorities. Weekly takes a different approach:

> *Plan your week on Sunday. Execute. Reflect. Repeat.*

Each week you get a clean slate with **7 days** and a **focus area** for the week. You drag tasks into days, check them off as you go, and start fresh next week. No clutter, no overdue piles — just a simple rhythm.

## Features

- 📅 **7-day grid** — one column per day, always showing the current week
- ✅ **Quick task entry** — add a task to any day in seconds
- 🎯 **Weekly focus** — define your one big goal for the week
- 🔄 **Weekly reset** — archive last week and start fresh
- 💾 **Local storage** — everything saved in your browser, no account needed

## Running Locally

Just open `index.html` in any modern browser — no build step required.

```bash
open index.html
```

## Tech

Plain HTML, CSS, and vanilla JavaScript. Zero dependencies.
