Skill: UI – Sistema de Colores

Filosofía: Paleta minimalista basada en violetas pastel y blanco hueso. Evitar colores saturados o contrastes agresivos.

Fondo claro: No usar blanco puro (#FFFFFF). Usar off-white / hueso (#F5F2F8 o similar).

Fondo oscuro: No usar negro puro (#000000). Usar gris violeta oscuro (#1C1A24 aprox.).

Color primario: Violeta pastel (#BFA8E6) para acciones principales y elementos interactivos.

Acento: Violeta más profundo (#8E79C9) solo para énfasis o estados activos.

Texto: Evitar negro puro. Usar gris violeta oscuro (#4A4458 en claro, #ECEAF3 en oscuro).

Tokens: Los componentes deben usar tokens semánticos (background, surface, primary, textPrimary, etc.), nunca hex directos.

Modo oscuro: El color primario se aclara ligeramente para mantener visibilidad (#CBB7F0 aprox.).

Jerarquía: Máximo 1 color dominante + 1 acento por pantalla.

Consistencia: No introducir colores nuevos fuera de la gama violeta definida.
